Installing
  1. Install Python
  2. Download console app
  3. Unpack the archive Messedges.zip
  4. Open a command prompt in the folder Messedges:
    • cd Messedges
  5. Install the dependencies:
    • Windows: pip install -r requirements.txt
    • macOS/Linux: pip3 install -r requirements.txt
  6. Run the app:
    • Windows: python run.py
    • macOS/Linux: python3 run.py
Usage
  • Limits
    • 1 room per IP
  • Room data available on https://messedges.com/room/<room-name>
  • Messages available on https://messedges.com/room/<room-name>/messages
